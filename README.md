# Drone Navigation Sim

> Autonomous drone pathfinding simulation with OpenAI Gym

## Overview
This project is part of a comprehensive AI/ML portfolio covering reinforcement learning python.

## Features
- Production-ready implementation
- Well-documented code with examples
- Easy to customize and extend

## Installation

```bash
git clone https://github.com/jaychampaneri14/drone-navigation-sim.git
cd drone-navigation-sim
pip install -r requirements.txt
python main.py
```

## Requirements
See `requirements.txt` for full dependencies.
Main libraries: Reinforcement Learning, Python, Robotics

## Usage
```bash
python main.py
```
Run with `--help` for full options.
